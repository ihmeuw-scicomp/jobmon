Getting Started
***************

Welcome to Jobmon! This section will help you get up and running quickly.

Jobmon is a Scientific Workflow Management system that simplifies running 
computational workflows on distributed computing systems. Whether you're 
running a few jobs locally or thousands of tasks on a cluster, Jobmon 
provides the tools to manage complexity, handle failures gracefully, and 
monitor progress.

What You'll Learn
=================

- **Overview**: Understand what Jobmon does and its key features
- **Installation**: Get Jobmon installed in your environment  
- **Quickstart**: Create and run your first workflow in minutes
- **Your First Workflow**: A detailed walkthrough building a real workflow

Choose Your Path
================

- **New to Jobmon?** Start with the :doc:`Quickstart <quickstart>` guide to get a 
  workflow running in under 5 minutes.
- **Want the full picture?** Read the :doc:`Overview <overview>` to understand Jobmon's 
  architecture and capabilities.
